PONG

Score:

Para jugar usa las teclas -W- -S- para mover la paleta izquierda arriba y abajo respectivamente.

Y usa las teclas -Arriba- -Abajo- para mover la paleta derecha arriba y abajo respectivamente.

Para iniciar o pausar el juego, usa la barra espaciadora

Para reiniciar el juego, usa la tecla -R-, y dale play con la barra espaciadora
